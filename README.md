## SetUp
### Cloning

* In your terminal:
        
        $ git clone ``

## Running the Application

* To run the application, in your terminal:

        $ pip install -r requirements.txt
        $ chmod +x run.py
        $ ./run.py
        
## Testing the Application

* To run the tests for the application file:

        $ python3 manage.py test
